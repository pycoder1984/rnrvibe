# Getting Started with Vibecoding

This guide will walk you through everything you need to start vibecoding today. No prior programming experience required.

## Step 1: Choose Your Tool

For beginners, we recommend starting with one of these:

- **Claude.ai** — Free tier available, great at explaining code
- **ChatGPT** — Widely used, good for general coding tasks
- **Cursor** — If you want a full code editor experience

Pick one and create an account. You can always switch later.

## Step 2: Start with a Simple Project

Don't try to build the next Facebook. Start small:

- A personal homepage
- A to-do list
- A simple calculator
- A countdown timer

These projects are small enough to complete in one session but complex enough to teach you the basics.

## Step 3: Describe What You Want

Open your chosen tool and write a clear description. Here's an example:

```
Build me a personal homepage with:
- My name centered on the page in large text
- A short bio paragraph below it
- Links to my GitHub and Twitter profiles
- A clean, minimal design with a white background
- Use HTML and CSS only
```

## Step 4: Review and Understand

When the AI generates code, don't just copy-paste it. Take a moment to:

1. **Read through it** — Even if you don't understand everything, get familiar with the structure
2. **Ask questions** — "What does this CSS property do?" or "Why did you use a div here?"
3. **Run it** — See the result in your browser

## Step 5: Iterate

Your first result probably won't be perfect. That's fine! Tell the AI what to change:

- "Make the text bigger"
- "Add a dark mode toggle"
- "Change the font to something more modern"
- "Add some spacing between the sections"

Each iteration teaches you more about how code works and how to communicate with AI effectively.

## Step 6: Save and Share

Once you're happy with your project:

1. Save the files to your computer
2. Consider using GitHub to store your code
3. Deploy it for free using Vercel or Netlify
